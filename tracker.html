March Madness 2026 Model Tracker

ML predictions vs actual results, updated in real time

How to Update

Edit the ACTUAL_RESULTS object in this file. For each game, add the actual winner's name. The tracker recalculates everything automatically.

Format: "Round of 64": { "0": "Duke", "1": "Ohio St.", ... } where the number is the game index (shown in each row).

Model: Stacked Ensemble (LR + GB + RF + HGB) | 73.6% holdout accuracy | 10,000 simulations
